# Nginx Exporter for Prometheus

This is a simple server that periodically scrapes nginx stats and exports them via HTTP for Prometheus
consumption.

To run it:

```bash
go run nginx_exporter [flags]
```

Help on flags:
```bash
go run nginx_exporter --help
```
